Adding a new blog

For each blog...
	Update blogs/index.php
	Record name of blog and url for blogger entry
	Add rss feed to opml file
	Add link to http://www.google.com/coop/
	-- Add to mailing list - http://alice.hatters.org.uk/cgi-bin/mailman/admin/brighton-bloggers/members/add
	Email owner to tell them that they are added
	
Once all added
	Post blog about new blogs added		
	Validate opml file at http://feedvalidator.org/
